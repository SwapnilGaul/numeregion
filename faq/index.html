Troubleshooting

I am having trouble installing TaraNG-19.1

Make sure you are downloading the latest version. Also make sure you have all prerequisite softwares and system dynamic libraries installed on your system.

I am having trouble finding system MAC address

You can find out MAC address from IP configuration using command prompt or terminal. You can get IP configuration using command
E.g.: 

ipconfig/all

I get an error - "Unable to start TaraNG"

First, make sure you have a valid license for the corrosponding version of TaraNG. Please generate license request, this option as it allows us to recreate the license. Restart TaraNG with the new license issued. Please contact us to get the license file.

I believe there's a bug in TaraNG.

First, make sure you are using the latest version of TaraNG. If the bug is still present, please submit an issue to us. We would be happy to resolve the problem and make the efficient solution available for you after everything is fixed and tested
